# BankCore Pro — Система автоматизации банковских операций 🏦

Проект представляет собой разработку архитектуры базы данных для ядра банковской системы (Core Banking) на базе **MS SQL Server**. Система поддерживает управление клиентской базой, мультивалютными счетами, кредитными и депозитными продуктами, а также логирование подозрительных операций.

## 🛠 Технологический стек
- **СУБД:** Microsoft SQL Server
- **Язык:** T-SQL
- **Инструменты:** SSMS, GitHub

## 🏗 Архитектура базы данных
База данных спроектирована с учетом требований целостности данных и включает следующие модули:
- **Clients & Branches**: Учет клиентов и филиальной сети банка.
- **Accounts & Cards**: Управление счетами и привязанными к ним картами.
- **Transactions**: Журналирование переводов и платежей.
- **Loans & Deposits**: Модули для работы с активами и пассивами (расчет процентов и остатков долга).
- **Security**: Таблица `FraudLog` для регистрации подозрительных транзакций.

## 🚀 Реализованные бизнес-задачи и SQL-навыки

В рамках проекта (`select's BankCore.sql`) решены следующие аналитические задачи:

1. **Анализ ликвидности**: Группировка балансов по валютам и филиалам банка (`JOIN`, `GROUP BY`).
2. **Сегментация клиентов**: Автоматическое присвоение категорий (Basic/Standard/VIP) в зависимости от остатка на счете (`CASE` выражения).
3. **Кредитный мониторинг**: Расчет фактического остатка задолженности путем сопоставления выданных сумм и истории платежей (`LEFT JOIN`, `ISNULL`).
4. **Финансовая прогностика**: Расчет ожидаемой доходности по активным депозитам за год.
5. **Безопасность и активность**: Выявление высокоактивных клиентов с оборотом более 500 единиц (`HAVING`).
6. **Работа с типами данных**: Форматирование банковских реквизитов и дат (`CONVERT`, `CAST`).

## 📁 Структура репозитория
- `createBankCore.sql` — DDL скрипт (создание таблиц и связей).
- `Insert's BankCore.sql` — DML скрипт (наполнение тестовыми данными).
- `select's BankCore.sql` — Бизнес-логика и аналитические отчеты.
